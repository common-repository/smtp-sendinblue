== Description ==
**Send emails from WordPress through Sendinblue using SMTP by YayCommerce**

SMTP for Brevo, formerly Sendinblue, helps you send sleek email notifications successfully. It has an advanced **email log** with various options to help you show/hide columns, view/filter data, and set up auto-delete old data as well.

With a good SMTP plugin dedicated to Sendinblue, you can make sure to improve the audience engagement and deliverability of your transactional email messages.

It works great for plain text, HTML mail, promotional and marketing email.

== Frequently Asked Questions ==

= Is this plugin compliant with GDPR? =
Absolutely! YaySMTP doesn't collect or store any personal information. So rest assured.

= Is YaySMTP free? =
Yes, you can use it free of charge. YayCommerce WP SMTP plugin for Sendinblue comes with all features that you can find in the description above.

= I want to switch from Brevo, what should I do? =
You can install our [YaySMTP primary plugin](https://wordpress.org/plugins/yaysmtp/). It comes with 15+ SMTP email services and allows you to [export/import email logs](https://docs.yaycommerce.com/yaysmtp/settings-and-tools/view-email-log).

= How can I further customize my emails? =
You can customize WooCommerce email templates by using [YayMail](https://wordpress.org/plugins/yaymail/) free plugin. Fully compatible and optimized for WooCommerce.

== Installation ==
1. Upload the entire plugin folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the **Plugins** menu in WordPress.
